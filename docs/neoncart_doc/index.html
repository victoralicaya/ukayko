Installation Guide

Note : You are advised to read the installation procedure carefully, backup your store files and database first before installation or making any changes.
Note : Please Create the Database before Installation. The name of Database will be required when asked during Installation.

The theme can be installed in two 2 ways

  1. Quickstart Installation : If you want to test the theme and check how the live demo was configured, you can install demo containing the sample data and settings from our live demo.
  2. Theme Files Installation : If you already have a Zencart store and want to apply the theme to that store. For the store running Zen Cart v157 or v157x, please follow these steps.

Quickstart Installation

  1. Quickstart Theme Installation is only for the users who want the latest Zencart version i.e v158
  2. Unzip the main downloaded package and find the QuickstartPackage-ZCv158 folder.
  3. Upload the content of this directory to your server using the FTP client or Cpanel
  4. Start installation by typing the URL of the directory where the files have been placed.
    (Ex: localhost/ [directory name] /installation/index.php or domain name/ [directory name] /installation/index.php)
    The initial installation page should appear as normal Zencart installation and then follow the next steps as shown below :

Database Setup

This step requires the Database information for Installation. Image below shows the Steps mentioned.

  • Enter your Database Host Name. (Ex: localhost)
  • Enter your Database Username. (Ex: root)
  • Enter your Database Password. (As per your requirement)
  • Enter your Database Name. (Ex: zencart)
  • Leave Store Identifier (Table Prefix) field empty.
  • Click on "Continue". This will take you to the next step of installation process.

Admin Setup

  • Enter the admin information as per your requirement and click on Continue.
  • A temporary Admin Password will be created and the admin directory will be renamed automatically as per the new Zencart installation process.

Setup Finished

  • This will complete the installation process and the page with both Admin and Store Front links will appear as shown below.
  • Now go to your installation directory and rename/delete the zc_install directory.
  • Click on Store Admin URL and it will ask to refresh the page again and a normal zencart admin login page will open. Enter the Admin Username and Temporary password. This will prompt you to change the temporary password as per the normal zen cart installation process

Store Setup

  • When you logged into you store admin Panel and you can will two alert messages on top of your store admin as shown in the image below. Install the Related Products Module and Neoncart Template Settings by clicking on Click to Install link.
  • Please enter the store detail as shown in above image.



Theme Files Installation

  1. Unzip the main downloaded package and find the Neoncart Theme package folder as per your current Zen Cart version.
  2. Rename the YOUR-ADMIN-FOLDER-NAME directory with the name of your existing store admin directory.
  3. Now copy all the files and directories of the ThemePackageFiles to your existing store directory. This will replace some files in your existing store.
  4. Login to you Store Admin Panel and you will see two alert messages on top of your store admin panel as shown in the image below. Install the Neoncart Template Settings by clicking on Click to Install link.


  5. Once that is done you will see the Template has been installed successfully. message on top and will be redirected to the Theme Admin Panel.
Important: Please note that as authors of this theme we are able to provide support only for the features which we created and for the issues related strictly to this theme. We do not provide support for Zencart configuration, installation, maintenance, customization etc.

Templates Settings

Neoncart template can be customized according to customer's need. Template Settings makes customization very easy and hassle free.
Once the Template is installed completely, Template Settings menu can be found under Tools > Neoncart Template Settings in admin area.
Following are the features that can be customized using Neoncart Theme Panel.

  • General

    Go to Tools > Neoncart Template Settings > General > General Settings Panel section to start your custom settings configuration.


    General Page Layout > Here you can choose options for the page layouts.

    Page Loader > Here you can choose options for display Page Loader. If you want to set custom page loader images you can set custom option.

    Image Loader > Here you can Enable - Disable Images Loader.

    Container Width > Here you can set container width.

  • Color

    Go to Tools > Neoncart Template Settings > General > Color Panel section to start your custom settings configuration.


    Theme Color > The default theme color is set to #cc1414, you can enter the color code of your choice or select the color from the color picker provided.

    Theme Second Color > The default theme color is set to #0062bd, you can enter the color code of your choice or select the color from the color picker provided. Second color affected like button hover background color.

    New Label Color > The default theme color is set to #cc1414, you can enter the color code of your choice or select the color from the color picker provided. New Label color affected to new product label color.

    Special Label Color > The default theme color is set to #0062bd, you can enter the color code of your choice or select the color from the color picker provided. Specials Label color affected to special product label color.

  • Fonts Settings

    Go to Tools > Neoncart Template Settings > General > Fonts Settings Panel section to start your custom settings configuration.


    General Font Family > Neoncart comes with 500+ Goolge fonts preloaded. The General Font family will affect all the fonts in theme excluding the banner cations that you have added. Default: Open Sans

    Heading Font Family > Neoncart comes with 500+ Goolge fonts preloaded. The Heading Font family will affect all Heading, Title etc. Default: Oswald

    Banner Heading Font Family > Neoncart comes with 500+ Goolge fonts preloaded. The Banner Heading Font Family will affect all banner heading etc. Default: Poppins

    Character Set: Latin Extended > Only selected fonts support extended character sets. For a complete list of available fonts and font subsets please refer to Google Web Fonts.

    Custom Character Subset > Only selected fonts support character sets. For a complete list of available fonts and font subsets please refer to Google Web Fonts. eg: greek,greek-ext

  • Product Slider

    Go to Tools > Neoncart Template Settings > General > Product Slider Panel section to start your custom settings configuration.


    Products Slider Show Columns > Show the numbers of products that you set in specific size of screen width

    Additional Image Style > This option comes with 2 different values. Selecting default option will remove the additional images from the products and will have a simple view.

    If you want to display Alternate Images, it can be done by selecting Hover Effect option. Once the Hover Effect option is selected, another option below it will appear to let you choose the style of display on hover display alternative image display.

    In these other options to you can set Hide-Show products elements in product slider.

    Products Slider options affected all products slider like micro slider, tab slider, product slider etc...

  • Testimonial Manager

    Go to Tools > Neoncart Template Settings > General > Testimonial Manager Panel section to start your custom settings configuration.


    Display Testimonials > Select Yes or No as per your requirement to display the Testimonials on your store home page.>

  • Social Links

    Go to Tools > Neoncart Template Settings > General > Social Links Panel section to start your custom settings configuration.


    Enter the Social links info in the text boxes provided. As of now Theme supports Facebook, Twitter, Pinterest, Google Plus and Instagram for your store. This links will be displayed on the header top bar, footer column and on contact us page. Leave the text box empty to remove that particular social link.

  • Newsletter

    Go to Tools > Neoncart Template Settings > General > Newsletter Panel section to start your custom settings configuration.

Disable Language/Currency Dropdown

  • Language and Currency Dropdowns are shown in the top bar of theme. You can disable them if you do not require in your store. Follow the below steps to do so :
  • Login to your store Admin Panel and go to Configuration > Layout Settings.
  • At the bottom of page you will find option Show Languages in Header? for languages and Show Currencies in Header? for currencies. Both the values are set to "True" by default, change the value to "False" to remove it from Top Bar/Header.

News Box Manager

  • Neoncart comes with the News Box Manager plugin preinstalled. The configuration settings can be found under Admin > Configuration > News Box Manager. You can set the Items to show on Home page and for Sidebox too. The News content lenght can also be changed using these options.
  • If you are willing to add new News for your store, go to Admin > Localization > News Box Manager where all your current news will be displayed. You can also insert/add latest news story for your store using this option.

  • Best Seller products are displayed on the Home page in Products Tab.

  • Best Seller products image height/width can be set using from this section.
